## .prettierrc

```
printWidth: 한 줄에 표시할 최대 문자 수
tabWidth: 탭의 크기
useTabs: 탭 대신 스페이스 사용
semi: 문장 끝에 세미콜론 추가 여부
singleQuote: 작은따옴표 사용 여부
trailingComma: 객체나 배열 끝에 쉼표를 추가할지 여부
bracketSpacing: 객체 리터럴에서 중괄호 사이의 공백 추가 여부
arrowParens: 화살표 함수 매개변수에 괄호 추가 여부
```
